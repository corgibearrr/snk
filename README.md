# 샷건 & 카타나 (Shotgun & Katana)

탑다운 트윈스틱 슈터. HTML5 단일 파일 게임.

## 실행 방법
`index.html` 을 브라우저(Chrome / Edge / Firefox) 에서 열면 바로 실행됩니다.
> ⚠️ 일부 브라우저는 로컬 이미지 로딩에 보안 제한이 있습니다.
> 만약 캐릭터/스탠딩 CG가 안 보이면, 폴더에서 간단한 로컬 서버를 띄우세요:
> ```
> python -m http.server 8000
> ```
> 그 후 `http://localhost:8000` 접속.

## 조작
- **WASD** : 이동
- **마우스** : 조준 (조준선 + 산탄 범위 표시)
- **좌클릭(짧게)** : 샷건 3발 발사
- **좌클릭(꾹 누름)** : 0.5초마다 1발씩 충전 → 떼면 일제 발사 (최대 12발)
- **우클릭** : 카타나. 누르고 있으면 1→2→3단계 충전 (흰→노랑→황금). 떼면 휘두름. 투사체 반사 + 적 베기 (데미지 12)
- **스페이스** : 구르기 (무적, 배터리 소모)
- **E** : 캐롯마켓 상점 열기 (BTC로 업그레이드)
- **\` (백틱) 3번 빠르게** : 치트 메뉴 (BTC +99999 / 배터리 +999 / 한계돌파 강제 발동)
- **ESC** : 일시정지 / 모달 닫기
- **M** : 음악 ON/OFF

## 게임 흐름
1. 시작 시 탄약 120/360, 배터리 100, BTC 0.
2. 적이 사방에서 몰려옴. 적을 죽이면 확률적으로 탄약(10%) / BTC(30%) / 배터리(3%) 드랍.
3. **3분(인게임 시간)마다 보스 등장**. 빨간 경고 메시지 표시. 보스 처치 전까지 일반 적은 안 나옴.
4. 보스 처치 → **한계돌파** 메뉴 (3개 랜덤 업그레이드 중 1개 선택).
5. **5단계(제미네이터)** 처치 시 엔딩.

## 보스
| 단계 | 이름 | 특징 | HP |
|---|---|---|---|
| 1 | 백규 | 6연발 리볼버 + 회피 굴리기 | 20 |
| 2 | 크랙슨 | 일직선 돌격 방패병 (벽 충돌 시 경직) | 40 |
| 3 | 리퍼 | 벽 무시 저격수, 순간이동 (한방에 죽음) | 1 |
| 4 | CP-09 | 장애물/적 소환 + 포격 해커 | 50 |
| 5 | 제미네이터 | 사족보행탱크, 뒤쪽 빨간 약점 공격, 탄막 + 추적포격 | 400 |

## 한계돌파 9종
전자기총탄 / 런앤건 / 건카타 / 조준경 / 슬라이딩부츠 / 관통탄 / 축지 / 라이브러리안 드론 / 시체폭발

## 캐롯마켓 상점 9종
탄약구매(BTC 50, +30발) / 배터리구매 / 기본 산탄수+1 / 최대 충전수+3 / 최대 탄약+30 / 이온충전기 / 이온배터리슬롯 / 카타나 데미지 / 카타나 충전속도

업그레이드는 레벨이 올라갈수록 가격이 가파르게 비싸집니다.

## 이미지 교체 방법
`images/` 폴더 안의 PNG 파일들을 **같은 파일명** 으로 바꾸면 됩니다:
- `title.png` — 타이틀 화면 로고 (현재는 HTML 텍스트로 렌더링되므로 비워둬도 무방)
- `standing.png` — 우하단 스탠딩 CG (300×480 권장)
- `player_idle.png` — 캐릭터 대기 (64×64, 위쪽이 정면)
- `player_move.png` — 캐릭터 이동
- `player_shoot.png` — 캐릭터 사격
- `player_slash.png` — 캐릭터 칼 휘두르기

이미지가 없으면 게임 내부의 도형 렌더링이 자동으로 표시됩니다.

## 사운드
- **BGM**: 빠른 템포의 헤비메탈 (180 BPM) 이 Web Audio API 로 절차적 생성됨. 외부 파일 불필요.
- **SFX**: 사격/장전/베기/충전/피격/픽업/폭발 모두 절차 생성.
- HUD 우상단 ♪ 버튼 또는 M 키로 토글 가능.

> 만약 외부 음악 파일을 쓰고 싶다면, `sounds/bgm.mp3` 등을 추가한 후 `game.js` 의 음악 함수에서 `<audio>` 태그로 교체할 수 있습니다.

## 임시 이미지 재생성
